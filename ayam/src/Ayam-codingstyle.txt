o line length max 79 characters
o indention: Emacs-C-mode-style (2 chars), except local variables and
  last return in a function (just 1 char), glBegin()-glEnd(),
  RiBegin()-RiEnd() pairs indent one character
o all {} are on their _own_ line (resembles the look of Pascal code a bit)
o Ayam symbols and typenames are all preceeded by "ay_"
o all preprocessor definitions are preceeded by "AY" and are in ALL CAPS
o function naming convention: ay_[modulename]_getblurb[type]
  type: "cb": Ayam callback, "tcb": Togl callback, "tcmd": Tcl command
o variable naming convention (not mandatory) ay_varname[type]
  type: "d": double, "i": int, "p": pointer, "t": ayam table 

example:
