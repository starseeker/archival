This directory contains a pretty much trimmed down version of libsub
from the Subdivide2.0 package by Henning Biermann and Denis Zorin
( http://mrl.nyu.edu/~biermann/subdivision/ )
with additional interfaces for Ayam.

Removals include: all (tag) flat mesh functionality, manipulators,
Inventor and VRML interfaces.
